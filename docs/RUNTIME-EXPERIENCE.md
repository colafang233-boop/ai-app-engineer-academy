# Runtime Experience Guide

## Start

```bash
python3 -m http.server 8000
```

Open:

```text
http://localhost:8000/apps/runtime-academy/
```

## Two modes

### Learning mode

- Lessons unlock sequentially.
- Column 1 exam requires all four lessons.
- Passing Column 1 unlocks Column 2.
- Column 2 exam requires all five lessons.

### Experience mode

Turn on the top-right `体验模式` switch to open any lesson or exam without changing the lesson content. This is intended for product review and interaction inspection.

## Suggested review route

1. Lesson 03: apply four code repairs and run diagnostics.
2. Lesson 04: add the necessary history to the context window.
3. Lesson 06: move `refund_query` above `complaint` and rerun the table.
4. Lesson 07: compare type error, business conflict and valid JSON.
5. Lesson 08: choose boundary, contrast and failure examples.
6. Lesson 09: remove the bad missing-order rule and rerun evaluation.

## Artifact ledger

Open `产物账本` from the top bar. Each successful simulator writes a real artifact used by later lessons.
